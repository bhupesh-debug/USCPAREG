## 🎯 What Is the CPA REG Exam?

✅ **Think of it this way:** The government wants to make sure CPAs (Certified Public Accountants) know the rules about taxes, laws, and ethics. The REG exam tests you on all of that. It's like a big "do you know the rules?" test for accountants.

### Exam Format
| Detail | Info |
|--------|------|
| Time | 4 hours |
| MCQs | 72 multiple-choice questions (50% of score) |
| TBSs | 8 task-based simulations (50% of score) |
| Sections | 5 testlets |
| Passing Score | 75 |

## 🧠 Master Mnemonic: **"REG = Rules, Entities, Governance"**

✅ **Think of it like this:**
- **R**ules = Tax rules for individuals and property
- **E**ntities = How businesses (C-corps, S-corps, partnerships) get taxed
- **G**overnance = Ethics, law, and professional responsibilities
